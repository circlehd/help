# Video Player controls

![](../.gitbook/assets/video-conrol.png)

### Keyboard shortcuts

| Key | Functionality |
| :--- | :--- |
| Space | Play pause |
| F | Full screen |
| Left key | forward 10s |
| Right key | rewind 10s |
| M | Mute/unmute |
| UP | Volume up |
| Down | Volume Down |
